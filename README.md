# 🎧 YouTube Music Inspired Prototype

A frontend music player prototype built using HTML, CSS & JavaScript.

## 🚀 Features
- Interactive music player bar
- Click on song card to update player
- Live search functionality
- Dynamic rendering using JSON
- Horizontal scrolling shelves

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
